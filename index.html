Course Planner

A C++ application designed to manage university course catalogs using both CSV files and a SQLite database. The program allows students to browse courses, view detailed information, filter by major, and track prerequisites.

Key features include:

  • Loading course data from CSV files or a SQLite database
  • Displaying sorted course lists and detailed course information, including prerequisites
  • Filtering courses by major to create customized schedules
  • Admin login system with authentication for managing course data
  • CRUD operations (Create, Read, Update, Delete) on courses and prerequisites
  • Use of software design patterns like Singleton and Iterator for efficient data and state management

This project demonstrates skills in C++ programming, database integration, software design principles, and modular application development. Emphasis was placed on code readability, maintainability, and proper separation of concerns, making it easy to extend the system with additional features.
